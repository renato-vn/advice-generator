# Random Advice App

This app, built with React.js, TypeScript, and TailwindCSS, displays a random advice each time the user clicks a dice-shaped button. The central card shows a title "Advice #{number}" with the advice text below. The dice button triggers a new random advice to inspire users.

## Features

- Random advice display on button click
- Interactive dice button as trigger
- Clean, responsive UI centered on screen
- Built with React.js, TypeScript, and TailwindCSS

## Installation

1. Clone the repo

`git clone`

2. Install dependencies

`yarn`

3. Run the app

`yarn dev`

## Usage

Open the app in a browser. Click the dice button to get a new random advice displayed in the card center.
